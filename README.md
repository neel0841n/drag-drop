# Drag&Drop — Free File Converter

A fast, private, browser-based file conversion tool. No uploads. No server. No sign-up.

## 🌐 Live Demo

Open `index.html` directly in your browser — or deploy to GitHub Pages in one click.

---

## ✅ What It Does

All conversions run **100% in your browser**. Files never leave your device.

| Tool | Works In Browser |
|---|---|
| Merge PDF | ✅ |
| Split PDF | ✅ |
| Compress PDF | ✅ |
| PDF to JPG | ✅ |
| JPG to PDF | ✅ |
| Rotate PDF | ✅ |
| JPG to PNG | ✅ |
| PNG to JPG | ✅ |
| Resize Image | ✅ |
| Compress Image | ✅ |
| Word / PPT / Excel to PDF | ⚙ Needs backend |

## 🚀 Deploy to GitHub Pages (Free Hosting)

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://yourusername.github.io/draganddrop`

---

## 🛠 Tech Stack

- Pure **HTML + CSS + JavaScript** — zero frameworks
- [pdf-lib](https://pdf-lib.js.org/) — PDF creation and editing
- [PDF.js](https://mozilla.github.io/pdf.js/) — PDF rendering to images
- [JSZip](https://stuk.github.io/jszip/) — batch ZIP downloads
- **Canvas API** — image conversion and resizing
